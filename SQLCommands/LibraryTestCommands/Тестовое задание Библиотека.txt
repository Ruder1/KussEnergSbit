
ЗАДАЧА
  Спроектируйте и создайте структуру БД для каталога библиотеки (SQL скрипт).
  Обязательные реквизиты: Название книги, автор, год издания.
  Дополнительные реквизиты - на ваше усмотрение.
  Загрузите набор тестовых данных (SQL скрипт).
  Прокомментируйте созданную структуру БД.

  Загрузите в созданную базу тестовые данные (SQL скрипт).

  Выполните следущие запросы:
    Получить список книг, написанных автором N (включая соавторство, не включая 
      соавторство).
    Получить список книг, написанные автором N в соавторстве с M.
    Получить список книг, которые написаны 3-мя соавторами.
    Получить всех авторов, у которых вышло более одной книги за указанный год.

ПОДСКАЗКА
  Автор может иметь несколько книг.
  Книга может иметь несколько авторов.
  Книга может иметь несколько изданий разных лет (с разными наборами авторов).

РЕАЛИЗАЦИЯ
  СУБД - Предпочтительно Oracle. Можно использовать бесплатную версию Oracle Express Edition.
         Но можно и другую СУБД, с поддержкой ANSI SQL.
  Язык - SQL.
